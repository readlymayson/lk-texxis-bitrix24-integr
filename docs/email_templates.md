# Шаблоны email-сообщений для регистрации в Личном кабинете

## Общая информация

Все шаблоны используются при создании ЛК для контакта (раздел 7.5 ТЗ).

### Структура файлов

Шаблоны доступны в двух форматах (BBCode и HTML) и сохранены в отдельных файлах:
- **Проектировщик**: `email_templates/designer_bbcode.txt`, `email_templates/designer_html.html`
- **Интегратор**: `email_templates/integrator_bbcode.txt`, `email_templates/integrator_html.html`
- **Торговый Дом**: `email_templates/trading_house_bbcode.txt`, `email_templates/trading_house_html.html`

### Переменные Bitrix24 (используются в HTML шаблонах)

HTML шаблоны используют переменные Bitrix24 для автоматической подстановки данных:

- `{{Имя}}` - имя пользователя (контакта)
- `{=Template:URL}` - ссылка для авторизации в ЛК
- `{=A60732_67341_77361_88578:USER_LAST_NAME}` - фамилия ответственного менеджера
- `{=A60732_67341_77361_88578:USER_NAME}` - имя ответственного менеджера
- `{=A60732_67341_77361_88578:USER_EMAIL}` - email ответственного менеджера
- `{=A60732_67341_77361_88578:USER_PERSONAL_MOBILE}` - телефон ответственного менеджера

**Примечание:** BBCode шаблоны используют старые плейсхолдеры с символом `#` и требуют ручной замены.

## Примечания по использованию

1. **BBCode** - используется в системах, поддерживающих BBCode форматирование (например, некоторые форумы, CMS системы)
2. **HTML** - используется для отправки HTML-писем через email-клиенты
3. Все плейсхолдеры (начинающиеся с `#`) должны быть заменены на реальные значения перед отправкой
4. В HTML версии добавлены базовые стили для корректного отображения в email-клиентах
5. Ссылки оформлены как кликабельные элементы в обоих форматах
