# Скрипт test_get_company_inn.php

## Назначение

Скрипт для тестирования получения ИНН компании из системных реквизитов Bitrix24 через API `crm.requisite.list`.

## Использование

```bash
php test_get_company_inn.php [company_id]
```

## Примеры

```bash
# Проверить компанию с ID=4
php test_get_company_inn.php 4

# Проверить компанию с ID=6
php test_get_company_inn.php 6
```

## Что проверяет скрипт

1. **Получение данных компании** - проверяет, что компания существует в Bitrix24
2. **Получение ИНН из реквизитов** - вызывает метод `getCompanyINN()` из `Bitrix24API`
3. **Детальная информация о реквизитах** - показывает все реквизиты компании для отладки

## Вывод

Скрипт выводит:
- ✓ УСПЕХ - если ИНН найден
- ⊘ РЕЗУЛЬТАТ - если ИНН не найден
- Детальную информацию о всех реквизитах компании
- Время выполнения запроса

## Возможные результаты

- **ИНН найден**: Скрипт показывает ИНН и время выполнения
- **ИНН не найден**: Возможные причины:
  - У компании нет реквизитов
  - Реквизиты не содержат поле RQ_INN
  - Ошибка доступа к API

## Логирование

Все действия логируются в файл, указанный в конфигурации (`config/bitrix24.php`).
